Accéder à la plateforme

Depuis n'importe quel appareil connecté à internet

🔗 Comment accéder ?

Ouvrez le lien fourni par le responsable dans votre navigateur (Chrome, Safari, Firefox, Edge). Aucune installation ni compte requis — le formulaire s'ouvre directement.

💻 Sur ordinateur

Collez le lien dans la barre d'adresse. La sidebar de navigation est à gauche, le formulaire à droite.

📱 Sur téléphone

Le site s'adapte aux petits écrans. Appuyez sur ☰ en haut à gauche pour ouvrir le menu.

💡 Conseil : Ajoutez le lien en favori pour y accéder rapidement depuis le terrain.

📋 Étapes de remplissage

1
Saisissez le numéro de dossier Champ obligatoire — sans lui, l'enregistrement est bloqué.
2
Choisissez le type d'intervention Un des 4 boutons : Intérieur, Piscine, Toiture ou Extérieur. Le formulaire s'adapte automatiquement.
3
Cochez les réponses & renseignez les champs Parcourez les sections. Les cases cochées apparaissent en bleu marine. Saisissez noms, téléphones et observations dans les champs texte.

✏️ Champ "Autre"

Lorsque vous cochez une option "Autre", un champ de saisie apparaît automatiquement.

👉 Vous devez alors préciser la situation en renseignant les informations demandées dans ce champ.

4
Cliquez sur "✦ Enregistrer le questionnaire" Bouton vert en bas de page. Seules les infos connues au moment de l'appel sont nécessaires.
3

Enregistrer & Consulter la liste

Sauvegarde en temps réel, accessible par tous les gestionnaires

✅ Enregistrement réussi

Message vert en bas à droite : "Questionnaire enregistré !". Le formulaire se vide et est prêt pour le suivant.

❌ Erreur d'enregistrement

Message rouge. Vérifiez que le numéro de dossier est bien renseigné et que vous êtes connecté à internet.

🗑️ Bouton "Effacer"

Remet tous les champs à zéro sans enregistrer. Une confirmation est demandée avant l'effacement.

💡 Temps réel : Tous les gestionnaires voient les mêmes questionnaires instantanément.

≡ Accéder à la liste & rechercher

Cliquez sur Tous les questionnaires dans la sidebar. Le tableau affiche tous les dossiers du plus récent au plus ancien.

🔍 Recherche

Tapez un numéro de dossier, un nom ou un téléphone — les résultats se filtrent instantanément.

🏷️ Filtre par type

Boutons Intérieur, Piscine, Toiture, Extérieur pour n'afficher qu'un seul type d'intervention.

4

Exporter en PDF & Questions fréquentes

Générer un document imprimable · Résoudre les problèmes courants

⬇️ Générer le PDF

1
Allez dans "Tous les questionnaires"
2
Trouvez le dossierVia la recherche ou le filtre.
3
Cliquez sur "⬇ PDF"Bouton dans la colonne Actions à droite de la ligne.
4
Sélectionnez "Enregistrer en PDF"Dans la boîte d'impression. Sur iOS : Options → Enregistrer en PDF. Sur Android : choisissez "Enregistrer en PDF" dans les imprimantes.
⚠️ Popups bloqués ? Si rien ne s'ouvre, autorisez les popups pour ce site dans les paramètres de votre navigateur.

🔴 "Enregistrer" ne fonctionne pas

Vérifiez que le Numéro de dossier est rempli (seul champ obligatoire), puis vérifiez votre connexion et rechargez.

🔴 Questionnaire enregistré avec une erreur

Dans Tous les questionnaires, supprimez le dossier avec ✕ (définitif, confirmation demandée), puis recommencez.

🔴 Affichage incorrect sur téléphone

Utilisez Chrome ou Safari. Appuyez en dehors de la sidebar pour la fermer. Tournez en mode paysage pour plus de confort.

🔴 Je ne vois pas le questionnaire d'un collègue

Rechargez la page dans Tous les questionnaires. Les données sont partagées en temps réel.

🟢 Faut-il créer un compte ?

Non. La plateforme est accessible directement via le lien — aucun identifiant ni mot de passe.

🟢 Les données sont-elles sauvegardées si je ferme ?

Oui, à condition d'avoir cliqué sur Enregistrer avant. Sans enregistrement, les données saisies sont perdues.

🟢 Peut-on utiliser la plateforme hors connexion ?

Le formulaire se remplit hors connexion, mais l'enregistrement et la consultation de la liste nécessitent internet.
